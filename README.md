# Form Filler

A powerful Google Forms automation toolkit featuring a high-speed submission engine, a modern graphical interface, and a flexible launcher system. Designed to support both casual users and power users, Form Filler combines ease of use with extensive customization options.

## Launcher

The launcher acts as the central entry point for the project, allowing users to quickly choose how they want to run the application. From a simple menu, users can launch either the graphical interface or the terminal version, as well as select between different backends and performance modes. Built-in file validation and startup diagnostics help catch configuration issues before the program runs.

## User Interface

The CustomTkinter-based GUI provides a modern, highly customizable experience for configuring and managing form submissions. Features include:

* Form scanning and automatic question detection
* Live submission statistics and analytics
* Multiple built-in themes and appearance presets
* Custom color, font, and layout customization
* Saved form link management
* Advanced answer validation controls
* Sleep prevention options
* Real-time logs and debugging tools
* Session tracking and export capabilities

The interface is designed to make complex automation features accessible without requiring command-line knowledge.

## Form Filler Engine

At the core of the project is the Form Filler engine, which supports multiple submission backends:

### HTTP Backend

* Extremely fast submission speeds using HTTP requests
* Concurrent asynchronous submissions
* Randomized browser fingerprints and user agents
* Advanced validation handling
* Dynamic answer generation
* Template token system for randomized responses

### Chromium Backend

* Browser-based automation for improved compatibility
* Normal, headless, and turbo performance modes
* Reduced CAPTCHA frequency compared to direct HTTP submissions
* Automatic browser management and optimization

## Template System

The project includes a powerful template engine capable of generating randomized data for text fields. Templates support:

* Random letters, numbers, and symbols
* Username generation
* URL-safe strings
* External text-file lists
* Cross-question answer references
* Variable length ranges

This allows highly dynamic and realistic response generation while minimizing repetitive answers.

## Cross-Platform Support

Form Filler supports Windows, Linux, and macOS, including platform-specific optimizations such as sleep prevention, process management, and performance enhancements.

## Goal

The goal of Form Filler is to provide a feature-rich, customizable, and user-friendly environment for form automation while remaining flexible enough for advanced users who need detailed control over submission behavior, answer generation, and performance.
